# Deep Work Bot

Бот для отслеживания сессий deep work с секундамером.

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Получите токен бота от [@BotFather](https://t.me/botfather) в Telegram.

3. Вставьте токен в файл `.env`:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_здесь
   ```

## Запуск

Активируйте виртуальную среду:
```
env\Scripts\activate  # на Windows
```

Затем запустите бота:
```
python main.py
```

## Команды

- `/start` - Приветствие
- `/begin_deep_work` - Начать сессию deep work
- `/end_deep_work` - Закончить сессию и показать время
- `/status` - Показать текущее время сессии

## Описание

Бот помогает фокусироваться на deep work, отслеживая время сессий. Используйте его для продуктивной работы без отвлечений.
